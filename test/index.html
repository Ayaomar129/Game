Hangman

Instructions

Follow these instructions to play the game effectively. Make sure to read them carefully!

  • A word is hidden, and each letter is shown as "_".
  • Guess one letter at a time.
  • If correct → the letter is revealed.
  • If wrong → a part of the Hangman is drawn.
  • 7 mistakes → Game Over.
  • Guess the whole word before 7 mistakes → You Win!
